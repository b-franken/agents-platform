# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-21

### Added
- Multi-agent platform with auto-discovery and HandoffBuilder orchestration
- Core library (agent-core): AgentConfig, factory, registry, middleware, scaffold CLI, validate CLI
- Demo agents: helpdesk, knowledge-agent, data-analyst, expense-approver
- Advanced agents: incident-triage (structured output), code-reviewer (regex analysis), infra-analyzer (HITL)
- A2A cross-framework demo (examples/a2a-demo/)
- MCP server example (examples/mcp-server/)
- Documentation: tutorial, concepts, troubleshooting, FAQ
- Terraform infrastructure (POC + enterprise environments)
- Docker Compose with Aspire Dashboard
- Azure Functions deployment
- DevContainer / GitHub Codespaces support
- CI/CD: lint, format, test, validate
- Setup script and Makefile
